# Local environment for jetpill 🚀

Docker-compose 🐳 to start some components on development enviroment.

- RabbitMQ
- Postgresql
- ScyllaDB

# How to?

## Up

It will start 🚀 all apps. 

```
docker-compose up -d
```

## Down

It will stop ✋🏾 and remove all containers. 

```
docker-compose down
```

## Stop

It will stop ✋🏾 all containers. 

```
docker-compose stop
```
